# Lost World

RPG de mundo abierto 2D top-down desarrollado en Unity 6 (C#). Inspirado en la
libertad de exploracion de Zelda BOTW y el diseno de mundo de Elden Ring.

## Sinopsis

Despiertas en la pradera de un reino sellado. Tres guardianes corruptos custodian
el paso hacia la zona maldita. Derrotalos para romper el sello y avanzar.

## Controles

| Accion    | Teclado            | Mando            | Tactil (Android)      |
|-----------|--------------------|------------------|------------------------|
| Mover     | WASD / flechas     | Stick izquierdo  | Joystick en pantalla   |
| Atacar    | Espacio            | Boton oeste (X)  | Boton de ataque        |
| Correr    | Shift              | Boton sur (A)    | Boton de correr        |
| Avanzar texto | Espacio / Enter | Boton sur (A)   | Tocar la pantalla      |

## Objetivo

Derrota a los tres guardianes de la pradera para romper el sello de la zona maldita.

## Plataformas

- Windows (ejecutable .exe)
- Android (APK)
